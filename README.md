# SARCHAT

A simple chat app built with the [libdirecpp](https://github.com/TheLastBilly/libdirecpp).

## Installation

### Requirements
* cmake
* g++
* ncurses
* direwolf

#### First clone this repo:
```
git clone https://github.com/TheLastBilly/sarchat
git submodule update
```

#### Compile and install sarchat:
```
cmake
make
sudo make install
```
#### Start direwolf:
```
direwolf
```

#### Run sarchat:
```
sarchat
```
